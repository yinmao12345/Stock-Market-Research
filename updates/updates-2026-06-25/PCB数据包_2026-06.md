## 节点：PCB（印制电路板）本体

### 基本信息
- **id**: `pcb`
- **name**: 印制电路板 (PCB)
- **下游锚点**: 终端应用（消费电子/AI服务器/汽车电子等，可暂作为根节点）
- **layer_updated**: 2026-06-25

### (1) 市场规模
- 全球 PCB（2024）：约 **735-750 亿美元**（同比增 5.8%）；2025 预计约 778 亿美元；2026 预计约 814 亿美元。
- 中国 PCB（2024）：约 **4121 亿元**（占全球约 56%-72%，口径不一）；2025 预计约 4333 亿元。
- **AI 服务器 PCB 细分**：2025 约 56 亿美元，2026 预计达 **100 亿美元**（中金）。这是核心增量。

### (2) CAGR（未来 3-5 年）
- 整体全球 PCB：2024-2029 CAGR 约 **5.2%**；中国约 4.3%。
- **AI/HPC 服务器 PCB**：2023-2028 年均复合增速达 **32.5%**（远超行业）。高盛预测 2026-2027 全球 AI 服务器 PCB 增速超 110%，传统 PCB 仅 3-5%。
- 18 层以上高多层板：2024-2029 CAGR 约 **15.7%**（行业最高）。
- 建议节点取 AI 服务器/高多层板口径，因这是投资焦点。

### (3) 价值量占比
- PCB 作为根节点（下游是终端整机），可暂不填 value_share，或注明"PCB 占 AI 服务器整机成本约 X%"待后续补。
- relative_to: 终端应用（暂留）

### (4) 缺口状态——底层趋势指标
- **inventory_cycle**：高景气。2025 以来需求持续向上，台股 PCB 行业 1-4 月累计收入同比+14.9%；龙头海外订单能见度达 8-10 个月。
- **penetration_rate**：AI 服务器 PCB 渗透率快速提升中，高多层板/HDI/载板高端占比持续扩大。AI 服务器普遍要求 20-30 层（普通服务器 8-16 层）。
- **capacity_utilization**：2025 国内 PCB 产能利用率高达 **92.8%**；普遍稼动率高企。高端产能释放滞后于需求。
- **supply_tightness**：偏紧（高端结构性紧缺）。AI 高端 HDI 及高多层板产能及原材料供应相对紧张，供需缺口持续存在。
- **推导建议**：tight（紧缺）。理由：稼动率 92.8% 高位+订单能见度长+AI 服务器超高增速+高端产能滞后，多指标共振。

### (6) 技术演进
- **stages**: 单/双面板 → 多层板 → 高多层板(18层+) → HDI(高阶/Anylayer) → 封装基板(IC载板)
- **current**: 高多层板与高阶 HDI 为当前 AI 主战场，向封装基板/先进封装延伸
- **note**: AI 服务器推动层数从 8-16 层升至 20-30 层（超算达 78 层）。下一代看点是 CoWoP（Chip-on-Wafer-on-PCB 先进封装），若 2027 量产明确将大幅提升市场预期。封装基板（FC-BGA/ABF 载板）打破海外垄断是高端核心增量。
- **risk_level**: 低（PCB 制造环节是工艺壁垒+客户认证壁垒，技术演进是渐进升级而非颠覆替代，龙头护城河随高端化加深）

## ⭐ ref 拼接说明（测试产业链延伸的关键）

PCB 的上游就是已经做好的 CCL。在 `pcb.json` 的 upstream 里，用 **ref 引用**把 CCL 整棵子树挂接进来，同时可补充 PCB 直接相关、但不在 CCL 树里的其他上游环节：

```jsonc
"upstream": [
  { "ref": "ccl" },                          // ⭐ 引用已建好的 ccl.json，自动挂接整棵CCL子树（含其下铜箔/树脂/玻纤布）
  {
    "id": "pcb_drilling_equipment",
    "name": "PCB钻孔/电镀设备",
    "metrics": { "value_share": {"value": null, "unit":"%", "relative_to":"PCB", "note":"制程设备，非材料成本，占比另计"} },
    "companies": [
      {"name":"大族数控","ticker":"301200.SZ","role":"龙头","purity":80,"logic":"国内PCB钻孔/曝光/检测设备主力"},
      {"name":"鼎泰高科","ticker":"301377.SZ","role":"主要参与者","purity":70,"logic":"全球PCB钻针龙头"}
    ],
    "upstream": []
  }
]
```

### CCL 占 PCB 的占比（补充到 ccl 节点或在 pcb 里标注）
- CCL 占 PCB 总成本：约 **27.3%-30%**（普通口径），AI 高多层板口径达 **35%-40%**。
- 这个数字应填在 ccl 节点的 `value_share`（relative_to: PCB），数据包一里已提供。
