# MLCC陶瓷粉体（介质粉体）产业研究

_以下基于截至2026年6月的公开信息整理。需特别说明：陶瓷粉体是MLCC的核心上游材料，本轮（2026年）行情中"利润池正在向上游迁移"，且叠加中国稀土出口管制这一独特变量，是当前市场焦点。投资有风险，以下不构成投资建议。_

---

## 一、概况

**1. 市场规模**

需先厘清口径——"陶瓷粉体"有狭义/广义之分，统计差异极大：

- **狭义（外销MLCC配方粉市场，即真正在专业市场交易的部分）**：规模较小。<cite index="37-1">2021年中国MLCC配方粉市场规模约25.82亿元，2022年降至24.1亿元，2023年约25.4亿元，2025年预计增至28.4亿元</cite>。全球外销市场据义柏资本测算<cite index="31-1">约116亿元（人民币）</cite>。
- **广义（含厂商自用、基础粉+配方粉+添加剂全口径）**：观研报告口径<cite index="36-1">2022年我国MLCC配方粉行业市场规模约1204亿元（含自用）</cite>——此口径偏大，因含大量内部自供折算。

需注意：<cite index="36-1">MLCC行业自产自销比例约30%，其余来自商业市场采购</cite>，且日本厂商粉体大多自用，故"可外销交易市场"远小于"实际消耗量对应的价值"。**对投资而言，更有意义的是外销配方粉市场，规模在百亿人民币量级（全球）。**

**2. 未来5年CAGR**

历史增速：<cite index="36-1">2013年以来我国MLCC配方粉产量复合增速达18.63%</cite>。前瞻看，受AI高容粉拉动，结构性失衡推动需求高增：<cite index="50-1">据QYResearch报告，MLCC纳米钛酸钡粉体总需求增速约17.5%，远超产能增速9.2%，2030年总缺口达7万吨，其中高端粉体缺口最严重（2030年缺口6.5万吨，缺口率59.1%）</cite>。**综合判断：行业整体CAGR约10%（参考陶瓷材料大盘），但高端纳米钛酸钡粉体细分赛道CAGR可达17%以上。**

**3. 在下游产品的价值量占比**

这是陶瓷粉体最关键的特征——**在MLCC成本中占比最高**：<cite index="33-1">陶瓷粉料在低容MLCC产品中占比20–25%，高容MLCC成品中占比高达35–45%</cite>。<cite index="41-1">高端纳米钛酸钡粉体占MLCC成本的30–40%</cite>。但需注意：MLCC粉体在再下游（如AI服务器整机BOM）中占比就被稀释到可忽略——<cite index="45-1">MLCC粉体在整个MLCC成本中占比不到40%，在整个AI服务器物料成本中占比可忽略不计</cite>。

**4. 竞争格局**

**全球：高度寡头垄断，日美主导。** <cite index="35-1">全球市场CR5达81%，核心份额被日美企业垄断，日本堺化学、美国Ferro市占率分别为28%、20%</cite>；<cite index="40-1">2023年前三大堺化学28%、美国Ferro 20%、日本化学14%，CR3高达62%</cite>。<cite name="31-1">CR3为62%，CR5为81%</cite>。村田则自产自用为主。

**国内：国瓷材料一家独大，但高端仍弱。** <cite index="45-1">国瓷材料国内市场份额超80%，全球份额约20–30%（部分口径），稳居全球第二，仅次于日本堺化学</cite>（注：另有口径称其全球份额约10%，差异源于统计边界）。<cite index="33-1">国瓷材料是继日本堺化学后国内首家、全球第二家成功运用水热法批量产销纳米钛酸钡粉体的厂家</cite>。其他参与者：风华高科、三环集团（自供为主，未大规模外销），方兴科技、凯盛科技及旗下安徽中创等。

**核心痛点——高端缺位**：<cite index="44-1">国瓷材料虽打破日本垄断成为全球第二家能量产高纯纳米钛酸钡的企业，但其全球高端粉体市占率仅约2%，核心短板在于缺乏日本厂商数十年积累的正向配方设计能力，只能做对标日系的标准化产品，无法提供差异化定制</cite>。

## 二、库存周期情况

**1. 当前所处环节：周期反转上行 / 涨价启动 + 利润向上游迁移**

陶瓷粉体作为最上游，其周期略滞后于MLCC成品但弹性更大。当前正处于**MLCC景气向材料端传导、粉体涨价启动**阶段。<cite index="42-1">随着MLCC景气度持续传导，利润池正在向上游迁移，国瓷材料的年内强势走势已透露这一预期</cite>。

需求侧出现历史性供需失衡信号：<cite index="44-1">国瓷材料合同负债在2025年已达5600万元，同比增长198%</cite>——这是订单前置、需求旺盛的直接证据。

**2. 最近一年（2025年中–2026年6月）逐月维度变化**

|维度|方向（趋势）|说明|
|---|---|---|
|**库存**|↓ → 低位|跟随MLCC去库尾声，粉体环节库存低位，高端供给紧张|
|**资本开支**|↑ 增加|<cite index="44-1">国瓷现有钛酸钡有效产能约1.5万吨，其中AI/车规高端粉5000吨，2026年还将新增3000吨高端产能</cite>；但扩产慢，<cite index="50-1">国瓷每年仅能扩约1000吨高端产能</cite>|
|**需求**|↑ 增加（加速）|AI高容粉拉动，合同负债同比+198%|
|**价格**|↑ 增加|<cite index="44-1">钛酸钡粉体随下游同步提价10–15%</cite>；<cite index="45-1">常规粉售价6–7万元/吨、车规8–9万元/吨、AI服务器用粉超10万元/吨</cite>|
|**开工率**|↑ 高位（高端满载）|高端粉产能优先放量、满产；常规粉相对宽松|

**关键独立变量——稀土管制（粉体环节特有，区别于MLCC成品）**：<cite index="50-1">2026年1月6日商务部发布对日本出口管制公告，中重稀土是高端MLCC粉体必要成分，日本除中国外无其他采购来源，已丧失扩产能力</cite>。<cite index="45-1">氧化钇（齿科氧化锆关键添加剂，也用于高端粉体）从2024年底不足8美元/公斤涨至超300美元/公斤，海内外价差超36倍</cite>。这给国瓷带来"原料稳定+海外减产让份额"的双重红利逻辑。

## 三、渗透率情况

陶瓷粉体本身无"渗透率"概念，最相关的代理指标是**高端钛酸钡粉体的国产化率（国产替代渗透率）**。

已知时点：<cite index="48-1">国内高端粉体国产化率不足10%</cite>（高端长期被日美垄断）；中低端已基本国产化。这是一个**国产替代刚起步、提升空间巨大**的环节。

## 五、技术演进路径

核心是**钛酸钡粉体的细化、均匀化、配方化**，紧密配合下游MLCC小型化/高容化：

1. **制备工艺升级（固相法→水热法）**：<cite index="33-1">固相法/草酸法可规模化但颗粒大、不均、品质低；溶胶-凝胶法最优质但易团聚、成本高；水热法颗粒细且均匀、成本相对低，业内普遍预测水热法将替代其他工艺</cite>。<cite index="31-1">全球能以水热法批量生产的仅日本村田（自用）、堺化学、国瓷材料</cite>。
    
2. **粒径细化（核心差距所在）**：<cite index="35-1">日本企业可做到80–100nm，国内目前仅能做到120–150nm</cite>。<cite index="37-1">村田可在D50约100nm钛酸钡基础上改性，制作0402/0201的10–100μF小尺寸MLCC，而国内厂家在D50约300–500nm基础上制作，存在差距</cite>。
    
3. **配方粉改性（最高壁垒、最保密）**：<cite index="31-1">通过添加钇、钬、镝等稀土保证绝缘性，添加镁、锰、钒、铬等保证温度稳定性和可靠性，添加剂须与钛酸钡均匀分布</cite>。<cite index="39-1">配方粉改性是各家独门绝技，保密性极高</cite>——这正是国瓷"技术领先却商业平庸"的症结（能做基础粉，缺正向配方设计能力）。
    
4. **晶型控制**：<cite index="39-1">四方相钛酸钡具自发极化，铁电、压电、热电性能更优，是市场需求重点</cite>。
    

**技术目标**：向200nm以下、乃至80–100nm高端粉突破，匹配AI服务器/车规高容、小尺寸需求。

## 六、终端应用与下游景气度

**应用链条**：陶瓷粉体 → MLCC（占绝大部分需求）/ 热敏电阻(PTCR)等 → 再下游消费电子、AI服务器、汽车电子、5G通信、工业、军工等。粉体需求几乎完全由MLCC景气度决定。

**当前下游景气度（强分化，高端极热）：**

- **AI服务器用高容粉 —— 极度高景气（核心驱动）**：直接受益于MLCC超级周期。<cite index="44-1">AI服务器对MLCC需求是普通服务器约8–12倍，单机柜MLCC价值量从H100的3000美元增至GB200的1.2万美元，Rubin VR200高达2.2万美元</cite>。AI/车规高端粉<cite index="44-1">毛利率可达45%以上</cite>，优先涨价、优先放量。
    
- **车规级粉 —— 高景气**：新能源车单车MLCC用量是燃油车4–6倍，车规粉售价8–9万元/吨，可靠性要求高。
    
- **消费电子用常规粉 —— 温和**：手机/PC需求平稳，常规粉售价6–7万元/吨，受高端挤压间接受益但弹性小。
    

**综合判断**：陶瓷粉体是本轮MLCC超级周期中"利润向上游迁移"逻辑的核心承载环节，且因稀土管制具备独立的供给收缩催化。<cite index="50-1">高端粉体结构性供需失衡明确，2030年高端缺口率达59.1%</cite>，中长期国产替代+高端突破空间巨大。

**风险**：①标的纯度低（国瓷粉体仅占其总业务一部分，高端粉仅占其产能不到7%），股价对MLCC涨价的反应更多是"预期推理"而非业绩直接映射；②估值已显著透支，短期回调风险高；③高端配方设计能力短板非短期可补；④若AI服务器出货不及预期或下游去库存，粉体作为最上游受冲击弹性也最大。
