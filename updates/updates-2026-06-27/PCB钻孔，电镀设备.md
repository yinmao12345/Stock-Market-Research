# PCB钻孔/电镀设备产业研究报告

> 数据说明：本报告以截至2025年底/2026年初的券商深度报告（东吴证券、东吴机械、西部证券、慧博投研、Prismark、QYResearch等）公开数据为准。部分细分（库存周期、月度数据、渗透率）行业不披露标准化时序数据，已按公开调研口径做合理推断并标注，请审慎使用。

## 一、概况

**1、市场规模（最新权威口径）**

按工序拆分，钻孔与电镀是PCB专用设备中价值量最高、工艺壁垒最大的两个环节：

|细分|2024年全球规模|国内规模|数据源|
|---|---|---|---|
|PCB钻孔设备（机械+激光合计）|约14.7亿美元（2020-2024从11.74亿增长至14.7亿）|约60亿元（2024，同比+7%）|慧博/QYResearch|
|—其中激光钻孔机|约10.0亿美元|—|QYResearch|
|PCB电镀设备|约5.1亿美元|垂直连续电镀(VCP)约24亿元|慧博/招股书|
|PCB专用设备（七大类合计）|70.85亿美元（国内41.11亿美元）|—|Prismark|

钻孔在PCB设备中价值量占比最大——2024年全球钻孔设备市场规模超100亿元、中国近60亿元（同比约+7%），在PCB设备中价值量占比最大。

**2、未来5年CAGR**

- 钻孔设备：2024-2029年市场预计进入加速增长阶段，规模将达23.99亿美元，CAGR提升至10.3%（前五年仅约5%）。激光钻孔机口径相对保守，QYResearch给出2023-2029年CAGR约5.9%（10.0→13.8亿美元）——差异主要在于是否计入AI驱动的高端机械钻/CCD背钻放量。
- 电镀设备：2024-2029年CAGR升至9.5%（2029年达约8.1亿美元）。

综合判断：在AI算力驱动下，**这两个环节2025-2029实际CAGR大概率落在10%以上**，且头部国产厂商增速远超行业（大族数控2024营收同比+104%）。

**3、在下游（PCB）价值量占比**

按PCB制造各工序对应设备价值量拆分：钻孔约21%（价值量第一），曝光约17%，电镀约7%，压合约6%。钻孔+电镀合计约占PCB专用设备价值量的**28%**，是设备投资的核心。

**4、竞争格局**

整体PCB设备格局分散：国内CR5市占率合计仅23.9%，大族数控国内第一占约10.1%（全球6.5%），高端PCB设备国产化率不足30%。分环节看高度集中：

钻孔设备——市场由德国Schmoll、日本日立、大族数控、维嘉数控和东台主导；大族在普通钻机市占率40-50%，高端CCD背钻由Schmoll和大族领跑（大族约5-10%）。高端超快激光钻仍由日本三菱、日立、德国Schmoll垄断，全球份额约90%，单台300-450万元。

电镀设备——三类路线分立：垂直连续式（VCP，东威科技占国内50%以上）、垂直升降式（台湾竞铭、东莞宇宙）、水平连续式（以德国安美特为代表）。东威是国产VCP绝对龙头，VCP国内市占率超50%，累计出货1200+台。

## 二、库存周期情况

**1、当前所处环节：主动补库存（景气上行加速期）**

行业已明确走出2022-2023年下行去库阶段。大族在2025年报中表示信息产业设备业务逐步走过去库存周期。2025年起进入由AI算力驱动的主动补库/扩产高峰：AI算力需求高速增长，传导至PCB设备厂商订单高度饱和。券商定性为"**业绩兑现元年**"——头部设备商订单已锁满2年，2026、2027连续高增长，2026Q3-Q4确认营收利润。

这是典型的**主动补库存**特征：需求强、订单满、价格挺、产能拉满、企业仍在扩产。

**2、最近一年（约2025年）逐月维度方向判断**

行业不公布标准化月度时序，下表为基于上市公司调研/财报的方向性归纳（↑增加 / →持平 / ↓减少）：

|维度|方向|依据|
|---|---|---|
|在手订单/需求|↑ 持续大幅增加|维嘉2025年1-8月出货超1800台、同比约+3倍，在手约800台，交期排到2026年3月|
|资本开支（扩产）|↑ 持续增加|大族二期基地投产；维嘉工厂满负荷、正建二期|
|产成品库存|→ / ↓ 偏紧|满负荷生产、交期拉长，库存周转加快、产成品偏紧|
|价格/单价|↑ 结构性上行|高价值CCD背钻、超快激光钻、Ⅲ类设备占比提升带动均价上行|
|开工率/产能利用|↑ 满载|工厂满负荷运转；大族2025年3月起产能超载、3月单月发货破百台|

唯一约束是供给端瓶颈：产能不足是首要瓶颈，主轴供应尤为紧张。

## 三、渗透率情况

PCB钻孔/电镀设备本身不是"渗透率"型产品，但其需求由下游PCB工艺升级渗透率驱动，核心指标为**高端工艺/材料在新增设备中的渗透**：

- **VCP占新增电镀设备比例**：长期稳定在约50%（该结构性比例已成熟，月度变化不显著）。
- **超快激光钻渗透率**：从近乎0快速起量。大族超快激光钻预计2025年实现批量交付，2025年处于0→个位数%的早期爬坡。
- **CCD背钻渗透**：随AI高多层板放量，2025年逐季提升。

由于仅有"2024年≈0、2025年开始批量"这类时点数据，按线性插值，**2025全年超快激光钻在激光钻新增中的渗透率大致由年初约1%线性升至年末约5-8%**（推断值，非披露数据，仅供趋势参考）。月度精确渗透率行业不可得。

## 五、技术演进路径

由AI高速、高多层、细线宽三大方向驱动，设备升级路径清晰：

**钻孔**：机械钻（孔径≥0.15mm）→ CCD背钻（高价值，单台130-140万 vs 普通钻48万） → 激光钻（孔径≤0.15mm）→ 从CO₂激光钻（80-150μm孔）演进至**超快激光钻**。超快激光核心优势是材料兼容性强（可加工Q布/玻璃），30-80μm微孔精细度更高。终极方向是**TGV玻璃基板钻孔**（先进封装），LPKF的LIDE激光诱导深度蚀刻为代表方案。

**电镀**：龙门式 → 垂直升降式 → **VCP垂直连续电镀**（当前主流）→ 脉冲VCP / 水平镀三合一 / **mSAP闪镀**。mSAP半加成工艺推动闪镀设备需求，实现超薄、快速、均匀打底电镀；脉冲电镀应对深孔镀层均匀度。

**材料/工艺催化**：M9Q布材料引入加快钻针磨损催生超快激光钻；Rubin服务器板厚升至6mm以上对40倍长径比钻针需求成为胜负手；1.6T光模块要求线宽缩至15μm驱动全工序升级。

## 六、终端应用与下游景气度

**终端行业**：AI服务器/数据中心（核心驱动）、高速交换机/通信设备、汽车电子、消费电子、低空经济（无人机）等。

**景气度：高景气、AI驱动的结构性爆发期。**

- AI算力是最强引擎：每台AI服务器PCB产值较传统服务器提升5-7倍；全球服务器/数据存储PCB市场2024-2029年CAGR达11.2%，AI/HPC服务器PCB 2023-2028年CAGR高达32.5%。
- 龙头业绩兑现：大族数控2025Q3营收15.21亿元同比+95%、归母净利2.28亿元同比+282%；行业平均PE-TTM达82.33，半年翻番，市场看好PCB处于爆发前夜。
- 产能/扩产持续：头部PCB公司2025-2026形成项目投资额419亿元，曝光、钻孔、电镀、检测环节最受益。

**核心结论**：钻孔/电镀设备是AI PCB浪潮中确定性最高的"卖铲人"环节，当前处主动补库+扩产兑现期，国产替代（高端钻孔、电镀）与价值量提升双轮驱动，景气度有望延续至2027年。
